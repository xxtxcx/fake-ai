# Fake AI Voice Mode Demo

Аудіореактивний fullscreen-інтерфейс у стилі voice assistant:
- 3D orb на `Three.js` з кастомним shader-матеріалом
- реакція вершин орбу на гучність мікрофона
- пульсація кольору від синього до фіолетового
- картка статусу та анімація `thinking`

## Як запустити

1. Відкрийте термінал у папці проєкту.
2. Запустіть локальний сервер (потрібен `https` або `localhost` для мікрофона):

```bash
node server.js
```

3. Відкрийте [http://localhost:8080](http://localhost:8080).
4. Натисніть кнопку **"Увімкнути мікрофон"** та дозвольте доступ.

## Налаштування під себе

- Швидкість анімації: `uTime` та обертання в `main.js`
- Сила реакції на звук: множники в `getAudioLevel()` і `vertexShader`
- Палітра: `uColorA` / `uColorB` у `main.js`

## Деплой на Vercel

Цей проєкт статичний, тому добре підходить для Vercel (і мікрофон працюватиме через HTTPS).

1. Створіть репозиторій на GitHub і завантажте туди файли проєкту.
2. Зайдіть на [Vercel](https://vercel.com), натисніть **Add New -> Project**.
3. Імпортуйте ваш GitHub-репозиторій.
4. Для налаштувань збірки:
   - **Framework Preset**: `Other`
   - **Build Command**: залишити порожнім
   - **Output Directory**: залишити порожнім
5. Натисніть **Deploy**.

Після деплою отримаєте URL виду `https://your-project.vercel.app` — відкривайте його на будь-якому пристрої та давайте доступ до мікрофона.

### Важливо

- `server.js` потрібен лише для локального запуску; на Vercel він не використовується.
- Якщо браузер блокує мікрофон, перевірте дозволи сайту (іконка замка біля адреси).

## Трансляція вашого голосу іншим

Реалізовано підхід, схожий на `obs-helper`: **serverless sync state на Vercel**.

- `source` (ваш ноутбук) відправляє аудіо-дані в `POST /api/orb-sync`
- `viewer` читає стан із `GET /api/orb-sync?since=...`
- усі глядачі бачать один і той самий рух орбу від вашого голосу

### Сценарій через Vercel (рекомендовано)

1. Задеплойте цей репозиторій на Vercel.
2. На ноутбуці відкрийте ваш Vercel URL, оберіть `source`.
3. У `Server URL` залиште той самий Vercel URL (або порожньо).
4. На інших пристроях відкрийте той самий Vercel URL, оберіть `viewer`.
5. Вкажіть однаковий `channel` (наприклад `demo-room`) і натисніть `Підключити`.

### Локальний режим (fallback)

`server.js` також підтримує локальну трансляцію через `/ingest` + `/events` (SSE), якщо не використовуєте Vercel API.
